# HTML Projects Collection

A collection of beginner and intermediate HTML projects created to practice and improve web development skills. These projects focus on building clean and responsive web pages using HTML and related technologies.

## Projects Included

- Personal Portfolio Website
- Landing Page
- Registration Form
- Survey Form
- Tribute Page
- Product Page
- Simple Blog Layout
- Image Gallery
- Resume Website
- Restaurant Website

## Technologies Used

- HTML5
- CSS3
- JavaScript (in some projects)

## Purpose

The purpose of this repository is to strengthen fundamental web development concepts and gain hands-on experience by building various projects.
